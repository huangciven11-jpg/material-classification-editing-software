# 素材分类剪辑软件设计文档

> 产品路径：`E:/codex/新建文件夹`
> 产品名称：素材分类剪辑软件
> 当前范围：第一阶段 MVP（素材资产中心 + 单版本粗剪生成）

## 1. 产品定位

这是一款面向短视频 / 电商视频生产的桌面端 AI 工具，不是普通 SaaS 后台，也不是传统复杂剪辑器。它的核心定位是：

- 素材资产中心
- 脚本驱动的半自动剪辑工作台
- 任务控制台

第一阶段不追求完整平台，而是先做成一个可真实使用的闭环：

**导入素材 → 一次性分析建库 → 搜索与人工校正 → 输入脚本 → 拆镜头位 → 候选召回 → 单版本粗剪 → 导出 mp4**

## 2. 第一阶段边界

### 2.1 本阶段包含

1. 批量导入素材
2. 素材入库时一次性分析
3. 内容/视觉/风格/用途标签化
4. 近重复聚类
5. 素材搜索、筛选、人工校正
6. 输入脚本并拆成镜头位
7. 为镜头位召回候选素材
8. 单版本粗剪生成
9. 导出 mp4
10. 真实任务进度显示

### 2.2 本阶段不包含

1. 5～10 条视频批量低重复生成
2. 完整配音上传转文字链路
3. BGM 智能匹配
4. 品牌风格模板
5. 工程文件导出
6. 完整多版本对比系统

## 3. 核心业务规则

1. 素材识别只在入库阶段执行一次。
2. 新素材只做增量识别，不全库重跑。
3. 电商素材不能只做内容粗分类，必须继续区分：
   - 构图
   - 景别
   - 角度
   - 背景
   - 光线
   - 手部动作
   - 配饰
   - 氛围
   - 用途
4. 素材匹配不能随机，必须采用：
   - 候选召回
   - 多维打分排序
   - 相似组规避
5. 所有耗时任务都必须显示：
   - 当前步骤
   - 当前文件
   - 百分比
   - 剩余时间
   - 成功/失败数量

## 5. 页面设计摘要

### 5.1 素材库页

目标：
- 导入素材
- 浏览素材
- 编辑标签
- 查看相似组
- 设置禁用 / 降权 / 收藏
- 查看使用记录

布局：
- 左侧：筛选与素材池范围
- 中间：素材缩略图主区
- 右侧：素材详情与编辑区

### 5.2 生成页

目标：
- 输入脚本
- 自动断句
- 拆成镜头位
- 查看每个镜头位候选素材
- 生成单版本粗剪

布局：
- 左侧：脚本输入与参数区
- 中间：镜头位卡片区
- 右侧：当前镜头位详情与候选替换区

### 5.3 版本页

目标：
- 预览当前生成结果
- 查看时间线片段
- 局部替换镜头
- 再导出

### 5.4 任务页

目标：
- 查看素材分析 / 生成 / 导出任务
- 查看失败原因
- 支持重试

### 5.5 设置页

目标：
- API 配置
- 标签体系
- 命名规则
- 导出参数

- 任务记录
- 导出记录

## 7. 核心数据对象

### Asset
素材资产对象，长期存在于素材库。

### AssetAnalysis
素材分析结果对象。

### SimilarGroup
近重复聚类对象。

### ScriptDocument
脚本文档对象。

### ScriptSegment
镜头位对象。

### CandidateAsset
候选素材对象。

### GenerationJob
生成任务对象。

### GeneratedVersion
生成结果对象。

### TimelineClip
时间线片段对象。

### TaskEvent
进度事件对象。

3. 近重复聚类
4. 素材库浏览与人工校正

## 9. 第二阶段能力

1. 多版本批量生成
2. 低重复控制
3. 配音上传与转写
4. BGM 智能匹配
5. 多版本对比
6. 品牌模板体系

## 10. 最终判断

第一阶段的目标不是做成“大而全 AI 视频平台”，而是做成一个真正可用的桌面端最小闭环工具：

**素材资产中心 + 脚本驱动的单版本粗剪工作台**

只要这条链稳定跑通，后续再加多版本、配音、BGM、模板，才不会回到半成品状态。
